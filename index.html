Data Types in Detail

1. Primitive Types

String

Text inside quotes.

let name = "Rahul";
let city = 'Mumbai';

Number

Integers or decimals.

let age = 21;
let price = 99.50;

Boolean

Only true or false.

let passed = true;
let failed = false;

2. Composite Types

Array

Ordered list, index starts at 0.

let fruits = ["Apple", "Banana", "Mango"];
fruits[0];  // Apple

Object

Key-value pairs.

let student = { name: "Priya", age: 20 };
student.name;  // Priya

3. Special Types

Null

Intentional empty value.

let x = null;

Undefined

Declared but not assigned.

let y;
// y is undefined
